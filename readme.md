# 초식은 무서워 (Scarely Herbivore)

📌 2D 전략 디펜스 게임 (Unity)

## 🎮 실행 방법
- [Windows 빌드 다운로드](https://drive.google.com/file/d/1OdMzf0Eas-UYeScjbhj5OrIoh9d7ivTs/view?usp=sharing) 후 실행

## 🛠 개발 정보
- Unity 5.3.4f1
- C#

## 📂 프로젝트 구조
- `Assets/` : 게임 리소스 및 스크립트
- `ProjectSettings/` : Unity 프로젝트 설정

## 👤 역할
- 팀 프로젝트 (5인)
- 본인 역할: 프로그래머
